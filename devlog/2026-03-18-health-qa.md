# Health Q&A Skill
**Date**: 2026-03-18  |  **Status**: Completed (v2 improved)

## What Was Built
A general health Q&A skill that answers questions about symptoms, conditions, medications, preventive care, mental health, and first aid by fetching credible information from the NIH/NLM MedlinePlus API. An LLM gate function (`needs_health_qa`) classifies whether a user query is a general health question, and if so, the skill extracts focused medical search terms, queries MedlinePlus, parses the XML response, and injects formatted reference data into the agent's context. Every response includes a medical disclaimer.

## Architecture
User query flows through the reactive agent loop in `agent.py`, which calls `SkillRuntime`. The pipeline:
1. **Gate**: `needs_health_qa()` (GPT-4o) decides if query is a general health question
2. **Term extraction**: `_extract_medical_terms()` (GPT-4o-mini) converts conversational queries into focused medical keywords (e.g. "what are the symptoms of strep throat?" → "strep throat")
3. **Search**: `search_health_topics()` calls MedlinePlus web service with extracted terms
4. **Parse**: `_parse_medlineplus_xml()` extracts titles, summaries, URLs, aliases, categories from XML
5. **Format**: `format_health_results()` creates structured markdown with medical disclaimer
6. **Inject**: Agent injects as `HEALTH REFERENCE INFORMATION` context; `instructions.md` governs LLM usage

## Technical Decisions
- **LLM-based search term extraction** (gpt-4o-mini) before MedlinePlus API — dramatically improves result relevance for conversational queries. Without this, "what are the symptoms of strep throat" returns Antibiotics/Tonsillitis pages instead of Strep Throat.
- **Skill boundary separation** — health_qa explicitly defers nutrition/diet queries to the nutrition skill (removed from both gate prompt and routing keywords) to prevent redundant activations.
- **Structure-preserving HTML stripping** — converts `<p>`, `<li>`, `<br>` to readable text formatting instead of producing a wall of text.
- **MedlinePlus API (no key required)** — free, authoritative NIH source, no new dependencies.
- **Summary truncation at 1200 chars** with word-boundary split to keep context window usage efficient.

## Known Limitations
- **English only** — MedlinePlus supports Spanish but this skill only queries the English health topics DB.
- **No drug interaction checking** — general Q&A only; specialized drug databases (RxNorm, DrugBank) would be needed for interaction alerts.
