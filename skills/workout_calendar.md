# Workout Completion Detection

Detect phrases like "finished my workout", "done for today", "completed today's exercise",
"just did my workout", "worked out today" and automatically mark the current day as
complete in the user's workout calendar.

Routing keywords: finished done completed workout exercise today gym training session
